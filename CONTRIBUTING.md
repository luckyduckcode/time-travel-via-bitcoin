# Contributing

Thanks for your interest in contributing. To keep collaboration smooth and professional, please follow these guidelines:

1. Fork the repository and create a feature branch from `main` (or `master`).
2. Write clear commit messages and keep changes focused.
3. Open a pull request with a concise description of what you changed and why.
4. Ensure experiments use only test data or data you explicitly own.
5. Follow the `CODE_OF_CONDUCT.md`.

For security issues, see `SECURITY.md`.
